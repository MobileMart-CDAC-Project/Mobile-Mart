# Mobile-Mart
MobileMart online mobile store 

Online Mobile Mart is a full-stack e-commerce web application built using React.js and Spring Boot. It provides secure user authentication with JWT and OTP-based email verification. Users can browse mobile products and post and read customer reviews. The application integrates the Razorpay payment gateway for secure online payments. An admin module manages products and orders, following a layered architecture with MySQL for scalability and maintainability.

**Tech Stack**

React.js (Vite), Bootstrap

Spring Boot, Spring Security (JWT)

MySQL

Razorpay
